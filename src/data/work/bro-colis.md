## Intro

Bro-Colis est une application mobile développée lors de mes études de Bachelor CDA. Le but fictif de cette application est de pouvoir proposer ses plats maison ou non à un réseau proche.
Bro-Colis est comparable à un système comme leboncoin ou vinted car les plats sont proposés tels des annonces.

### FireBase 

Pour cette application, j'ai utilisé Firebase pour gérer l'authentification (Authentication), les données (Firestore) et le stockage d'images (Storage).

### Liste des annonces

Cette page propose une liste des annonces de plats dans l'ordre croissant de date de création. Des filtres permettent une recherche plus précise, principalement sur la distance séparant l'utilisateur et l'annonce.

<img src="/assets/bro-colis/annonces.jpg" width="200">

### Lire une annonce

On peut bien évidemment lire une annonce, ainsi qu'envoyer une demande de réservation et commencer une discussion avec le cuisinier.

<img src="/assets/bro-colis/annonce.jpg" width="200">

### Publier un annonce

L'utilisateur peut publier son annonce avec une image du plat, une description, un titre, la composition pour indiquer les allergènes, la date de cuisine du plat, etc.

<img src="/assets/bro-colis/publication.jpg" width="200">

### Annonce sur une carte

Sur cette carte Open Street Map on peut parcourir les annonces disponibles autour de soi. Lors du clic sur un des points, le titre de l'annonce ainsi que la distance avec l'annonce s'affichent.

<img src="/assets/bro-colis/carte.jpg" width="200">
